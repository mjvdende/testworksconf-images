- [Vagrant](https://www.vagrantup.com/) + [VirtualBox](https://www.virtualbox.org/)

Given:

    $ git clone https://github.com/xebia/robot-framework-workshop.git 
    $ cd robot-framework-workshop

When:

    $ vagrant up

Then:

    start building!
